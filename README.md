# FN Key Default Set
##Change Default Behaviour of FN Key

### Parameters

"-h | --help : see this help"
"-d | --disable : disable fn_lock (permanent)"
"-e | --enable : enable fn_lock (permanent)"
"-t : temporary disable fn_lock"
